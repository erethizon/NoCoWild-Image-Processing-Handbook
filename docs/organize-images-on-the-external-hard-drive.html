North Country Wild Image Processing and Uploading Handbook

Chapter 3 Organize images on the external hard drive

Back in the lab, download all of the photos from each SD card onto the external hard drive called “NoCoWild”. Use the iMac which has a built in SD card reader on the back side of the monitor.

There are two main steps to organizing the photos:

  1. Archiving the files in the Raw folder

  2. Copying and renaming the files in the Processed folder

3.1 Step 1: Archiving the files

On the external hard drive (called “NoCoWild”), we keep a set of original images that we never1 modify. We organize files by calendar year. Depending on the year, the folder names may be different than what you see below and we may use a slightly different folder and file naming convention, though the basic steps are similar to what you see here.

Be sure to check with Erika to make sure you know what file and image naming conventions are currently in use.

Original files (if this is the first time they are downloaded from the SD card) go into the folder called “2021 Game Camera Photos” (or the appropriate folder for the particular year). Within that folder, they are placed into a subfolder called “Raw”, and, if appropriate, within the Raw folder, into subfolders by Round. Within the lowest-level folder will be a single folder named for each game camera & SD card combination (Figure 3.1).

Figure 3.1: Organization of the “Raw” folder containing the original, unmodified images files.

You may need to create these folders when you first upload the data. Please use leading zeros and make sure every camera number is recorded as 3 digits and every sd card number is recorded as three digits. For example, for images from Camera 24 SD card 5, you would have C024_SD005.

In some years, we also incorporated a site name into the file structure, so that folders for each camera are included in a folder for each site.

Again, these files are the originals and should NEVER BE ALTERED2.

3.2 Step 2. Copying and renaming the files

Once we have all of the photos archived in the “Raw” folder, we copy them to another folder called “Processed” and rename each image file according to its camera number and SD card and in some cases Round and/or Site (Figure 3.2). It is easiest to do this on the iMac.

Figure 3.2: “Raw” and “Processed” folders and example file names within each.

  • First, copy the “Raw” folder and paste it inside of the “2021 Game Camera Photos” folder (or similar).

  • Second, rename the copied folder “Processed”

For the image files in the “Processed” folder, your next task will be to replace the “IMG” or “EK” string in each image file name (which string will depend on the camera model from which the pictures were made) with a special string containing some combination of camera, SD card, Round, and possibly site. All of the pictures in the single folder will receive the same code, because they come from the same Camera and SD card combination. The flow chart for determining the proper code is as follows (Figure 3.3):

(ref:figure3) Image naming conventions depending on camera number, SD card number, and possibly deployment round and site.

Figure 3.3: (ref:figure3)

Once you have your folder pasted into the “Processed” folder, follow these steps to rename the files:

  1. Double click on the folder containing the photos (Figure 3.4). You will see that the default file names are assigned by the camera and are totally boring and easily confused with other image files.

(ref:figure4) File folder open showing original file names. Every camera will name images the same. We need unique names for images.

Figure 3.4: (ref:figure4)

  1. Now select all of the images in the folder. The fastest way is to use command-A.

  2. With all of the images selected, right-click the mouse and choose “Rename X items” where X is the number of selected files (figure 3.5)

Figure 3.5: Right-click to open the “Rename files” dialog box. Use “add text…” to add text into the file name.

Figure 6. When you right-click the image files you have selected, choose “Rename” to batch rename the files.

  1. A new dialog box will open (Figure 7).

In this example, the images all came from a forest called Donnerville, so I will enter “IMG” in the “Find” box and “DON” in the “Replace with” box, as in Figure 7.

  • Depending on what combination of camera, sd card, round and site you may be dealing with, you can replace the “IMG” with a full string indicating all of the information that needs to be changes. For example, instead of just replacing “IMG” with “DON” as in the example, you could replace it with something like “C024_SD002_R1”.

Figure 7. Find the “IMG” part of each filename and replace it with the appropriate combination of round and site codes.

Click the “Rename” button and the files will be renamed (Figure 8).

[]{#_Ref71037198 .anchor}Figure 8. You can see that all of the files have been renamed – the “IMG” was replaced by “BC” for each image.

  1. If you forget to add an element to a text name, (e.g. you forget to put the round number), you can use the "Add text" feature to do so. While all the files are still selected, right-click again and choose to rename the items again (i.e. repeat Step 3 above). This time, choose the dropdown arrow next to “Replace Text” and select “Add text”. Then choose the dropdown arrows next to “after name” and select “before name”.

When you are finished, you should see that your folder includes a set of renamed image files, like Figure 9:

[]{#_Ref71037215 .anchor}Figure 9. Appearance of files once correctly renamed.

Repeat these steps to save and rename any folders of images you import to the Processed folder on the hard drive.
